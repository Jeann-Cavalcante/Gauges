# Como executar o projeto

## Front end web
Pré-requisitos: npm / yarn

```bash
# clonar repositório
git clone https://github.com/Jeann-Cavalcante/app-dietas-react-native.git

# instalar dependências
npm install --legacy-peer-deps

# executar o projeto
npx expo start
```
